### NPM Packages
1. gulp
1. sass
1. gulp-sass
1. gulp-htmlmin
1. gulp-clean-css
1. gulp-image

### Step 1
```
Run the 'npm install' command to install the required NPM packages.
Also, install the above mentioned packages to devDependencies by running the command 'npm install --save-dev <package-name>'.
```

### Step 2
```
Open the index.html file (dist/html/index.html) with Live Server extension in VSCode to run the project.
```
